项目简介

案例以中国古典文学名著《红楼梦》为研究对象，运用社会网络分析（Social Network Analysis, SNA）中的图论方法，对小说中的人物关系网络进行系统性挖掘与分析。通过将文学文本转化为数学模型（图结构），量化计算各人物在社交网络中的核心度排名，从而揭示小说中隐藏的人物关系结构与社交格局。

方法与数据

本仓库目前以 HTML 页面为主，后续可加入数据文件（CSV/JSON）与分析脚本（Python/R）以及前端可视化脚本（D3.js / vis.js / sigma.js）。

  • 文本预处理：分词、人物实体识别、共现关系抽取
  • 图构建：节点为人物，边表示关系或共现，带/不带权重
  • 分析指标：度中心性、介数中心性、特征向量中心性、社区检测等
